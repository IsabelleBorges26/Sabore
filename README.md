## 📌 Sobre o Projeto
O **SABORÉ** é um site e aplicativo de receitas desenvolvido como Trabalho de Conclusão de Curso (TCC).  
A proposta do projeto é facilitar o acesso a receitas de forma prática, organizada e acessível, permitindo que usuários encontrem, salvem e compartilhem receitas.

---

## 🎯 Objetivos
- Criar uma plataforma de receitas intuitiva  
- Facilitar o acesso a diferentes tipos de receitas  
- Permitir interação entre usuários  
- Aplicar conceitos de desenvolvimento web e mobile  

---

## 🛠️ Tecnologias Utilizadas
- Frontend Web: HTML, CSS  
- Mobile: Flutter  
- Backend: Node.js  
- ORM: Prisma  
- Banco de Dados: MySQL  
- Ferramentas: Git, GitHub, VS Code
