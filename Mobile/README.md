# Mobile

## 📌 Descrição
Aplicativo mobile do sistema.

## ⚙️ Tecnologias
- Flutter
- Dart

## ▶️ Como executar

1. Acesse:
cd mobile

2. Instale:
flutter pub get

3. Execute:
flutter run

## 📱 Requisitos
- Flutter instalado
- Emulador ou celular

## 📂 Estrutura
- lib/ → código principal  
- screens/ → telas  

## ⚠️ Observações
- Backend deve estar ativo
