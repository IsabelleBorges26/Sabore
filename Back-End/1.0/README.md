# Backend

## 📌 Descrição
Responsável pelas regras de negócio, processamento de dados e comunicação com banco de dados.

## ⚙️ Tecnologias
- Node.js 
- Banco de dados (MySQL e Prisma)

## ▶️ Como executar

1. Acesse a pasta:
cd backend

2. Instale dependências:
npm install

3. Execute o projeto:
npm start

## 🔗 Endpoints
- GET / → Teste da API  
- POST /dados → Envio de dados  

## 📂 Estrutura
- controllers/ → lógica  
- models/ → banco  
- routes/ → rotas  

## ⚠️ Observações
- Configure o .env
- Banco de dados deve estar rodando
